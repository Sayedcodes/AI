## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Sayedcodes/AI.git
```

Go to the project folder:

```bash
cd AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
